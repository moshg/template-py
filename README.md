# template-py

Python template repository

## Requirements

* [uv](https://docs.astral.sh/uv/)

## Run app

Sync the project; e.g. install dependencies, etc.

```sh
uv sync
```


Run command hello.

```sh
uv run hello
```

## How create this template

Initialize the project

```sh
uv init --package
```

Edit ruff configuration in pyproject.toml.
